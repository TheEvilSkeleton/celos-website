CelOS - A powerful OS with minimal design

Flatpak is generally quite easy to learn and use, especially with the commands being similar to APT. (CelOS, Ubuntu and Debian's native package manager). The flathub repository for Flatpak is enabled on CelOS by default.

How to install applications:
flatpak install [app name]
Example: flatpak install discord

How to uninstall applications:
flatpak uninstall [app name]
Example: flatpak uninstall discord

How to search for applications:
flatpak search [app name]
Example: flatpak search discord
Alternatively you can use this website to find applications

How to update applications:
flatpak update [app name]
Example: flatpak update discord
